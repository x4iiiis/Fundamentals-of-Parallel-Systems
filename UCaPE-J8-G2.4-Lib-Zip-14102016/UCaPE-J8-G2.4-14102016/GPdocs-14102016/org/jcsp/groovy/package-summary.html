Package org.jcsp.groovy

Package org.jcsp.groovy Description

A set of Groovy helper class used to implement JCSP based parallel systems. JCSP is a library that uses Hoare's Communicating Sequential Process concepts to construct scalable networked parallel systems. The library was developed at the University of Kent by Peter Welch.
